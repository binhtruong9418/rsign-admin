## 1. Design Principles

### 1.1 Core Principles

**Progressive Disclosure:**

- Start simple, reveal complexity only when needed
- 5-step wizard breaks complex process into manageable chunks
- Smart defaults for common cases

**Visual Feedback:**

- Color-coded signers for easy identification
- Real-time progress indicators
- Clear success/error states
- Loading states for async operations

**Flexibility:**

- Templates for repeat use
- Quick actions for power users
- Full wizard for complex cases
- Keyboard shortcuts

**Error Prevention:**

- Validation at each step
- Can't proceed without required fields
- Clear error messages with solutions
- Confirmation dialogs for destructive actions

**Efficiency:**

- Auto-place signature zones
- Batch operations
- Template reuse
- Drag & drop everywhere

### 1.2 Color System

**Primary Colors:**

```css
--primary-blue: #2563eb;
--primary-blue-hover: #1d4ed8;
--primary-blue-light: #dbeafe;
```

## 6. Responsive Design

### 6.1 Breakpoints

```css
/* Mobile */
@media (max-width: 640px) {
}

/* Tablet */
@media (min-width: 641px) and (max-width: 1024px) {
}

/* Desktop */
@media (min-width: 1025px) {
}
```

### 6.2 Mobile Optimizations

**Wizard Steps:**

- Stack vertically on mobile
- One step per screen
- Swipe to navigate
- Bottom navigation bar

**Signature Zones:**

- Tap to place (no drag & drop)
- Full-screen zone editor
- Touch-friendly resize handles

**Dashboard:**

- Card-based layout
- Collapsible sections
- Bottom sheet for actions

---

## 7. Accessibility

### 7.1 WCAG 2.1 AA Compliance

**Color Contrast:**

- Minimum 4.5:1 for normal text
- Minimum 3:1 for large text
- Status colors meet contrast requirements

**Keyboard Navigation:**

- Tab order follows visual flow
- Focus indicators visible
- Escape closes modals
- Enter submits forms

**Screen Readers:**

- ARIA labels on interactive elements
- Alt text for images
- Role attributes for custom components
- Live regions for dynamic content

**Form Accessibility:**

- Labels associated with inputs
- Error messages announced
- Required fields indicated
- Help text available
